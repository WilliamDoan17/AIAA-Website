# Projects Idea

This is the idea for the Projects page for the AIAA - Club Website.

## Plan

### Main page 

  The main page for project for AIAA Website is going to be a page that shows current projects of the club as a list of cards, showing introductory information 

  It may classify projects and filter projects based on time tags || other categories 

  Goal: List out the projects and let people know what we're working on 
  
  #### Project Cards
  
  The Project Cards are cards that show the project information
  when clicked, it redirects the user to a new page which shows the info of the specific project 

### Project Detail Page (projects/project-id)

  This is the page that contains all the information of a specific project that the user choose to see
  
  It exists as a blog page, which the admin writes and post to the user to see

  Follows a default structure + Other structure by writer

### Execution 

- Start with the main page, which only list what project is current
- The Project Detail Page is gonna be planned in details when we reach the backend phase
