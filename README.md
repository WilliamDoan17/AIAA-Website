# AIAA-USF Website: The website for AIAA at USF 

## What the app is:
  This app is the interface to show information of the club AIAA, giving up-to-date club info about projects, events, and board members
  Later on will be a system for Admin to update information of the club as AIAA expand
## Who it is for:
  Aspiring and current club members
## Tech Stack:
  This project is currently being implemented using MERN Stack
