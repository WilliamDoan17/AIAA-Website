# PAGES

A list of application pages and their role in the AIAA-Website.

---

## auth
- `/login`: public, log in

## club_members
- `/admin/members/`: Admin-only. View, filter, invite, edit, or remove members.

---

## club_info
- `/admin/club`: Admin-only. See and edit club details.

---

## projects
- `/projects`: Public. List all projects with status and category filters.
- `/projects/:id`: Public. Project details, members, posts.
- `/admin/projects`: Admin-only. List, add, edit, and delete projects.
- `/admin/projects/:id`: Admins. Manage project members, posts.
- `/contributor/projects`: Contributors. View assigned projects.
- `/contributor/projects/:id`: Contributors. Manage posts, comment.

---

## events
- `/events/`: Public. List all events with time, location, and status.
- `/events/:id`: Public. Event details page.
- `/admin/events/`: Admin-only. Event list management.
- `/admin/events/:id`: Admin-only. View/edit event details.
